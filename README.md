# 📌 Overview

**AI Code Reviewer** is an AI-powered web application that analyzes JavaScript code and provides intelligent code review suggestions using the **Groq API**. It helps developers improve code quality by identifying potential bugs, improving readability, suggesting best practices, and recommending performance optimizations.

---

# ✨ Features

- 🤖 AI-powered code review using Groq API
- 💻 Interactive code editor
- 🎨 Syntax highlighting with PrismJS
- 📝 Markdown-based review output
- 📋 Copy AI Review
- 📄 Download Review as PDF
- ⚡ Fast REST API integration
- 📱 Responsive and modern UI
- 🔍 Real-time AI code analysis

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Ajaykumar187/AI-Code-Reviewer.git
```

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

### Backend

```bash
cd Backend
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the **Backend** folder.

```env
GROQ_API_KEY=YOUR_GROQ_API_KEY
PORT=3000
```
